{{ site.title }} {% gemoji zap %} is a CLI application written in Go to grab {% gemoji inbox_tray %} your files from a remote FTP or SFTP server to your NAS, server or computer {% gemoji computer %}. With Go, this app can be used across many platforms {% gemoji game_die %} and architectures. This support includes Linux, FreeBSD, macOS and Windows on architectures like amd64, i386, ARM and others.

Because FTPGrab is distributed {% gemoji package %} as an independent binary, it is ideal for those with a seedbox {% gemoji checkered_flag %} to grab your files periodically {% gemoji calendar %} to your Synology, Qnap, D-Link and others NAS.

It is free and open source and always will be {% gemoji unlock %} !
